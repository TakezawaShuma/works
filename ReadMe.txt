起動方法：
１：Server.exeを開く
２：TxiS.exeを複数枚開く


操作＆進行方法：
タイトル：Enterを押すと入力画面が出てきます。
	　PCのIPアドレスを入力して下さい。(ローカルの場合0000で入れます)
	　入力後もう一度Enterでシーンが変わりロビーに遷移します。

ロビー　：Enterを押すとシーン切り替え準備が完了します。
	　人数が複数人おり、全員が準備完了した場合シーンがプレイに遷移します。

プレイ　：矢印キー上下で前後移動
	　　　　　左右で自機の回転
	　スペースで弾を発射します。
	　左上のHPが全部なくなる、または敵を全員倒す、
	　または範囲外に一定時間以上出ることでシーンがリザルトに遷移します。

リザルト：Enterを押すとタイトル画面に戻ります。


備考：
通信対戦シューティングです。
ローカルホストなら正常に動くと思います。
複数台では正常に動かない可能性があります。
